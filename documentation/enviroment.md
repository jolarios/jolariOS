* [QEMU (emulador)](https://www.qemu.org/download/): `dnf install @virtualization`
* [limine (bootloader)](https://github.com/limine-bootloader/limine)
    * [OVMFbin](https://github.com/Absurdponcho/OVMFbin): firmware UEFI
